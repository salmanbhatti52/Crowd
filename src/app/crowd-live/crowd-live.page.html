CrowdLive® is a proprietary feature within the Crowd® App, developed by MiCrowd Ltd. The feature is designed to enhance your experience within the app, utilising data with a focus on privacy, transparency, and minimal data usage. Below, we outline how the feature utilises data, our commitment to data protection, and our approach to transparency with our users.
Data Filters and Limited Data Use
  • For CrowdLive® to function effectively, we collect a minimal amount of personal information, including the user’s name and profile image. This data is used strictly within the CrowdLive® feature to enhance visibility and interaction within the app in a secure manner.
  • Users' names and images are displayed only in designated CrowdLive® spaces and are not shared with any external parties or displayed outside the app.
Data Protection
MiCrowd Ltd is committed to maintaining the highest standards of data protection for all users of the Crowd® App. We use advanced security protocols and encryption methods to safeguard user information, especially when used by features like CrowdLive®.

  • Security Measures: All user data associated with CrowdLive®, including names and profile images, is stored securely on AWS servers. AWS provides industry-leading security standards, including multi-layered encryption and strict access controls to prevent unauthorised access.
  • Encrypted Storage: Personal information is encrypted both during transmission and at rest to ensure that sensitive data remains secure throughout its use within the app.
  • Location Privacy: Users' location data is securely encrypted on the backend and is only updated when the user enters a designated venue, adding them to the venue count within CrowdLive®. Outside of these locations, real-time location data remains private and inaccessible.
Transparency with Users
We prioritise transparency and want users to feel confident in how their data is managed within the Crowd App.

  • Data Visibility: Users can view all information that CrowdLive® uses, such as their name and profile image, within their account settings. Users retain control over this information and can make adjustments or updates as needed.
  • Limited Data Scope: We maintain a strict policy of only using data necessary to support the features within the Crowd App, with no additional or background data collection.
  • Opt-Out Option: Users may choose to disable the CrowdLive® feature if they prefer not to display their name and profile image within your profile, by enabling or disabling directly. Disabling this feature will remove the user from CrowdLive’s display list and will prevent any further use of their information for this feature. The user will then also be unable to view other users using the CrowdLive® feature to keep to a fair use policy.
